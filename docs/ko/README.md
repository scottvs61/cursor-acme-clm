# Knowledge & Operations (KO) Documents

Consumer-facing guides for each platform service. Use these when integrating with or operating the ACME server, CLM API, or SCEP server.

| Document | Service | Audience | Screenshots |
|----------|---------|----------|-------------|
| [KO-ACME-Server](KO-ACME-Server.md) | ACME v2 server | ACME client users (e.g. certbot), automation | — (API only) |
| [KO-CLM-API](KO-CLM-API.md) | CLM REST API + Web UI | API consumers, script authors, UI users | Yes (see [screenshots](screenshots/README.md)) |
| [KO-SCEP-Server](KO-SCEP-Server.md) | SCEP server | Device/MDM admins, SCEP client users | — (API only) |

**Screenshots:** The CLM KO includes placeholders for Web UI screenshots (certificate list, detail, bulk enroll, SCEP page). Add PNG/JPG files in `screenshots/` with the names listed in [screenshots/README.md](screenshots/README.md) and they will render in the KO.
